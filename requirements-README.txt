Για να τρέξει ο κώδικας πρέπει να έχετε μια εγκατάσταση της python3, να δημιουργήσετε ένα virtual environment 
και να εγκαταστήσετε τα παρακάτω πακέτα της python:

pip==24.0
protobuf==5.29.2
pysctp==0.7.2


Επίσης για να τρέξει το project θα πρέπει να έχετε στον ίδιο φάκελο τα αρχεία:
 
sctp_server
project_messages.proto
project_messages_pb2.py
client.py

Πρώτα τρέχουμε τον sctp_server ενώ εχουμέ συνδεθεί στο VPN του πανεπιστημίου 
και μετά εκτελούμε το script client.py που θα πραγματοποιήσει και την επικοινωνία client-server.
